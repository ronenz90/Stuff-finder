📷 יש רק תמונה? ניתוח ב-AI

ה-AI יהפוך את התמונה למחרוזות חיפוש מקצועיות וימלא את הטופס למטה אוטומטית.

הוראות שימוש

  1. פתח את כלי ה-AI החינמי שלך (Claude / ChatGPT / Gemini) וצרף את תמונת המוצר
  2. אם ברקע יש חפצים נוספים או רקע עמוס — הוסף שורה משלך אחרי הפרומפט, לדוגמה: "Focus on the silver cup in the center, ignore the tray behind it"
  3. אם ידוע לך מידע שלא רואים בתמונה (תקופה, מקור, יוצר) — הוסף גם אותו כשורה
  4. הדבק את הפרומפט, שלח, והדבק את תשובת ה-JSON בשדה למטה

הוספת מוצר למעקב

🔒 הכתובת לא נשמרת בקובץ הציבורי — היא מוצפנת ונשמרת כ-Secret בריפו. אם כתובת זהה כבר קיימת, המערכת תשתמש ב-Secret הקיים. השאר ריק לשימוש בכתובת הכללית (ALERT_EMAIL).

חיבור לריפו

הנתונים נשמרים בדפדפן שלך בלבד (localStorage) ולא נשלחים לשום מקום מלבד GitHub.

יצירה: GitHub ← Settings ← Developer settings ← Fine-grained tokens. תן הרשאות Read/Write על הריפו הזה בלבד ל: Contents (שמירת מוצרים), Actions (הרצת סריקה), Workflows (שינוי תדירות), Secrets (הצפנת כתובות מייל).
